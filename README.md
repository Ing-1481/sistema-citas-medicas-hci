# Sistema de Gestión de Citas Médicas

Prototipo desarrollado como parte del caso de estudio de la asignatura **Human Computer Interaction (HCI)** en la Maestría en Ciencia de Desarrollo de Software.

Este sistema permite gestionar citas médicas de forma sencilla mediante una interfaz web conectada a un servidor backend y una base de datos.

---

## Tecnologías utilizadas

- Node.js
- Express.js
- MongoDB
- HTML5
- CSS3
- JavaScript

---

## Funcionalidades del sistema

- Registro de citas médicas
- Visualización de citas registradas
- Edición de citas
- Eliminación de citas
- Validación para evitar duplicados
- Conexión con base de datos MongoDB
